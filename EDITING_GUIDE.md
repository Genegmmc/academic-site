# Editing Guide

## Main file

Edit:

```text
index.html
```

## Picture size

Search for:

```css
--photo-size: 300px;
```

Change only the number.

Smaller:

```css
--photo-size: 260px;
```

Larger:

```css
--photo-size: 340px;
```

## Replace photo

Replace:

```text
assets/profile.jpg
```

with a new photo using the exact same filename.

## Publications
